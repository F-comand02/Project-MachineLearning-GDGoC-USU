# Student Performance Prediction

## Description
This project predicts student exam scores using machine learning techniques.

## Dataset
Student Performance Factors Dataset

## Models Used
- Linear Regression
- Neural Network

## Best Model
Linear Regression (R² = 0.7696)

## Technologies
- Python
- Scikit-learn
- TensorFlow
- Streamlit

## Results
Linear Regression achieved the best performance with an R² score of 0.7696.
